
# 🌳 专注力金钱树 (Focus Gold Tree)

> **将你的深度专注，转化为看得见的数字财富。**

![React](https://img.shields.io/badge/React-19-61dafb.svg)

`Focus Gold Tree` 是一款专为提升专注力而设计的“游戏化”计时器。通过**复利增长模型**和**实时视觉反馈**，让你的每一秒专注都变得有价值。
一个可爱的 2D 小人会伴随你的专注不停挖掘，将金币源源不断地送入你的金库，挖出的金币会在屏幕一角堆成闪闪发光的金山，带给你直观的成就感。
---


## 🌟 核心理念：为什么选择它？

在信息碎片化的时代，深度专注 30 分钟的价值远高于 10 次 3 分钟的碎片时间。本项目通过数学公式模拟了这种价值：**专注时间越长，每秒赚取的金币就越多。**

## 🚀 核心功能

- **📈 复利引擎**：内置指数级增长算法。前 30 分钟收益缓慢，一旦进入深度专注，收益将呈爆发式增长。
- **🎮 游戏化体验**：观察勤奋的小矿工在金钱树下为你挖掘金币。
- **💎 实时财富看板**：高精度的计时器与财富金额显示，统一使用 `￥` 符号。
- **💰 自定义权重**：自由设定你的“时薪”，量化你的时间价值。

## 🧮 收益公式 (Economy)

应用采用以下公式计算即时收益：
$$Rate = \frac{HourlyWage}{3600} \times 0.1 \times (1.08^{minutes})$$

- **复利临界点**：约在第 30 分钟，收益率将达到你设定的 100% 时薪。
